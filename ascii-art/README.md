## Style Notes for Next Agent

- **Murals** are BIG art pieces (30+ lines) for major milestones — save to new files in this folder
- **Between-check doodles** are small (5-15 lines) and creative — vary them every time, don't repeat
- **Themes already used** (find NEW ones): dice, tanks, boats/armada, rockets, pyramids, skyscrapers, waterfalls, constellations, mountains, trains, tornado, braille rain, bar charts, burning joint, dice cups, dice tower, animal faces, linked tanks, arrow volley
- **Themes requested but not yet done**: DAB/oil rig (at 710 — user specifically asked for this), halfway mural (at 675)
- **Progress bars** always included in murals: `[████████░░░░░░] N/150 (XX%)`
- **Process count** always shown: `procs: N/12` or similar
- **Creative scenes** between murals — don't repeat, keep finding new metaphors
- The user got annoyed when art started repeating — **every piece must be unique**

## How to Continue

1. Read the current manifest at `trial-manifest.json` (in repo root)
2. Check current trial count

5. Save any new murals to this folder with descriptive filenames (e.g., `09-675-halfway-mural.txt`)
